# Data Structures & Algorithms

A collection of fundamental sorting and searching algorithms 
implemented in C, built as part of a university Data Structures course.

##  Algorithms Implemented

###  Sorting
| Algorithm | File |
|-----------|------|
| Heap Sort | `Heap_Sort.c` |
| Counting Sort | `Counting_Sort.c` |
| Insertion Sort + Quick Sort | `Insertion_Quick_Sort.c` |
| Binary Insertion Sort | `BIS.c` |

###  Searching
| Algorithm | File |
|-----------|------|
| Binary Search | `binarySearch.c` |
| Interpolation Search | `interpolationSearch.c` |

##  How to Run

Compile any file with GCC:

```bash
gcc Heap_Sort.c -o Heap_Sort
./Heap_Sort
```

Or use the provided `.exe` files directly (Windows).

##  Technologies
- C
- GCC compiler
